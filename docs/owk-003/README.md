# owockibot API Docs

This repository contains contributor-facing documentation for owockibot.

## Brazilian Portuguese Translation

The `pt-BR` directory contains Brazilian Portuguese translations for the contributor documentation used by the owockibot bounty board:

- [Visão geral do AI Bounty Board](pt-BR/README.md)
- [Diretrizes de envio de bounties](pt-BR/SUBMISSION_GUIDELINES.md)

## Source Scope

The translations preserve the formatting, code examples, API routes, wallet addresses, links, and technical terms from the upstream documentation in:

- `owocki-bot/ai-bounty-board/README.md`
- `owocki-bot/ai-bounty-board/SUBMISSION_GUIDELINES.md`

This PR is for bounty `owk-003`: translating the full owockibot contributor documentation into Brazilian Portuguese.
