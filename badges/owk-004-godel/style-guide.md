# owockibot Badge Style Guide

## System

The badges use one reusable visual grammar:

- Shape: round medal with an inner shield, readable at small sizes.
- Size: `256x256` viewBox for each standalone asset.
- Stroke: dark ink `#0f172a` with 8px primary strokes and 6px icon strokes.
- Palette: deep navy, teal, gold, coral, violet, sky, emerald, and slate accents.
- Typography: no embedded font dependency; text uses generic `Arial, sans-serif`.
- Accessibility: each SVG includes a `title` and `desc`.

| `badge-05-security-scout.svg` | Security Scout | Responsible security finding |
| `badge-06-api-builder.svg` | API Builder | Integration or API improvement |
| `badge-07-release-shipper.svg` | Release Shipper | Release support or deployment fix |
| `badge-08-mentor.svg` | Mentor | Helpful review or contributor support |

## Extension Rule

New badges should keep the same medal, shield, title, and icon structure. Change only:

- gradient colors
- central icon
- label
- small milestone marker

This keeps the collection recognizable while allowing new achievements.
