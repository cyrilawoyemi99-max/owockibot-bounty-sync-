# owockibot Contributor Badge System

Deliverable for bounty `owk-004`: 8 optimized SVG contributor badges with a shared style guide.

## Files

- `badge-01-first-merge.svg`
- `badge-02-bug-hunter.svg`
- `badge-03-docs-steward.svg`
- `badge-04-test-builder.svg`
- `badge-05-security-scout.svg`
- `badge-06-api-builder.svg`
- `badge-07-release-shipper.svg`
- `badge-08-mentor.svg`
- `index.svg`
- `manifest.json`
- `style-guide.md`
- `submission-note.md`

## Usage

Each badge is a standalone 256x256 SVG with no embedded raster images, scripts, fonts, or external references.

## Validation

From the repository root:

```bash
python scripts/validate-owk-004-godel.py
```
